# C3 Basics

This directory contains fundamental C3 concepts to get you started.

## Files

1. **01_hello_world.c3** - Your first C3 program
2. **02_variables.c3** - Working with variables and types
3. **03_functions.c3** - Defining and calling functions
4. **04_control_flow.c3** - If statements and loops
5. **05_arrays.c3** - Arrays and slices in C3

## How to Run

For any file, use:
```bash
c3c run <filename>.c3
```

Example:
```bash
c3c run 01_hello_world.c3
```

## Learning Order

Work through the files in numerical order. Each file builds on concepts from previous files.
