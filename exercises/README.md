# C3 Exercises

Practice problems to reinforce your C3 learning.

## Getting Started

Each exercise file contains:
- A problem description
- Function signatures to implement
- Example usage in `main()`

## Exercises

1. **exercise_01_fizzbuzz.c3** - Classic FizzBuzz problem
2. **exercise_02_fibonacci.c3** - Compute Fibonacci numbers
3. **exercise_03_string_reverse.c3** - Reverse a string
4. **exercise_04_array_sum.c3** - Array manipulation

## How to Work on Exercises

1. Read the problem description in the file
2. Implement the required function(s)
3. Run the file to test your solution:
   ```bash
   c3c run exercise_01_fizzbuzz.c3
   ```
4. Verify the output matches the expected results

## Tips

- Start with exercise 01 and work your way up
- Try to solve without looking at external resources first
- Compare your solution with different approaches
- Focus on writing clear, readable code

Happy coding!
