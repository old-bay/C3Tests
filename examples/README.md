# C3 Examples

Practical C3 programs demonstrating real-world usage.

## Files

1. **calculator.c3** - Simple calculator with error handling
2. **struct_example.c3** - Working with structs and methods
3. **file_operations.c3** - Reading and writing files
4. **dynamic_array.c3** - Using dynamic arrays and memory management

## Running Examples

```bash
c3c run <example_name>.c3
```

Each example demonstrates a practical application of C3 concepts.
